# API-gest-o-de-alunos-e-matriculas
Projeto desenvolvido com o objetivo de **estudar Spring Boot e APIs REST**, baseado em um **desafio técnico para vaga de backend júnior**. A aplicação implementa uma API para gerenciamento de **alunos e suas matrículas**, utilizando Java, Spring Data JPA e banco H2, aplicando conceitos básicos de backend.
